# Dashboard User Guide

## Accessing the Dashboard

### Power BI
Open the following file using Power BI Desktop:
dashboards/powerbi/ecommerce_analytics.pbix

---

## Dashboard Pages

### Page 1: Executive Summary
Provides a high-level overview of business performance.

**Key Metrics:**
- Total Revenue
- Total Transactions
- Average Order Value
- Profit Margin

---

### Page 2: Product Analysis
Analyzes product and category performance.

**Insights:**
- Top-performing products
- Revenue contribution by category
- Profitability analysis

---

### Page 3: Customer Insights
Focuses on customer behavior and retention.

**Insights:**
- Customer segmentation
- Repeat purchase trends
- Customer lifetime value indicators

---

### Page 4: Geographic & Trends
Shows regional and time-based sales patterns.

**Insights:**
- State-wise sales distribution
- Weekday vs weekend trends

---

## Filters Available
- Date Range
- Product Category
- Customer State
- Payment Method

---

## Refreshing Data
Run the pipeline and click **Refresh** in Power BI to load the latest data.

## Metric Definitions

- Total Revenue: Sum of sales amount
- Average Order Value (AOV): Total Revenue / Total Orders
- Customer Lifetime Value (CLV): Average Order Value Ã— Purchase Frequency
